Code School笔记 - Ruby Bits 2 Level 3

Self

在class外调用self会返回main，在class内调用会返回class本身。
如果在定义类中的方法时，在方法名前加self则这个方法会变成类似于Java的静态(static)方法

Class_eval

重新打开一个class并往里面增加一些东西

用 class_eval 写一个MethodLogger

就是记录方法的使用log

Instance_eval

类似于class_eval
